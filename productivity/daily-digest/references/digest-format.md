# Digest Format

## Standard Format
```
☀️ Good morning! Here's your [Day, Date] digest:

📰 TOP STORIES:
1. [Story summary] — @source
2. [Story summary] — @source
3. [Story summary] — @source

📈 MARKET:
- $BTC: $[price] ([%] 24h)
- $ETH: $[price] ([%] 24h)
- $SOL: $[price] ([%] 24h)

🔥 TRENDING:
- #hashtag ([X] mentions)
- "narrative" ([X] mentions)

💬 NOTABLE:
- @user: "[excerpt]"
- @user: "[excerpt]"

Have a great day! 🚀
```

## Minimal Format
```
☀️ [Date] Digest:

📰 [Story 1]
📰 [Story 2]
📰 [Story 3]

📈 $BTC $[price] [%] | $ETH $[price] [%] | $SOL $[price] [%]

🔥 [Trend 1], [Trend 2]

GM. ☕
```

## Degen Format
```
GM ANONS ☕

Here's what I'm seeing:

🚀 [Bullish thing 1]
🚀 [Bullish thing 2]
⚠️ [Risk/warning]

Portfolio check:
$TOKEN: [%] 🟢/🔴
$TOKEN: [%] 🟢/🔴

What are you watching today? 👇
```

## Thread Format
```
☀️ [Date] Daily Digest (1/5)

📰 TOP STORIES:

2/ [Story 1 summary]
3/ [Story 2 summary]
4/ [Story 3 summary]

📈 MARKET + 🔥 TRENDING:

5/ [Market data] + [Trends]

GM anons. What are you watching? 👇
```
