# Meme Prompts

## Classic Formats

### Drake Pointing
```
Top panel (rejecting): [bad thing]
Bottom panel (approving): [good thing]
```

### Distracted Boyfriend
```
Boyfriend: [current thing]
Girlfriend: [new exciting thing]
Other girl: [tempting alternative]
```

### Galaxy Brain
```
Level 1: [basic take]
Level 2: [smarter take]
Level 3: [smartest take]
Level 4: [enlightened take]
```

### This Is Fine
```
[Character sitting in fire]: "This is fine"
[Caption]: [describing a bad situation calmly]
```

### Trade Offer
```
I receive: [thing you want]
You receive: [thing they get / nothing]
```

### Expanding Brain
```
Small brain: [simple take]
Medium brain: [better take]
Large brain: [even better take]
Galaxy brain: [ultimate take]
```

## Crypto-Specific

### Not Losing Your Bag
```
[Person calmly doing something while everything burns]
Caption: "Me watching my bag drop 90%"
```

### Number Go Up
```
[Stonks meme with upward arrow]
Caption: "[Token] going up while I sleep"
```

### Wraith
```
[Wraith from World of Warcraft]
Caption: "I it" (for "I bought it")
```

### REKT
```
[Person looking at empty wallet]
Caption: "Me after buying the top"
```

### Diamond Hands
```
[Hodl meme]
Caption: "Paper hands sold. Diamond hands hold."
```

## Viral Formats

### Me Explaining to My Mom
```
[Person explaining something complicated]
Mom: [confused reaction]
Caption: "Me explaining [topic] to [person]"
```

### Nobody:
```
Nobody:
Absolutely nobody:
Me: [doing the thing anyway]
```

### How It Started vs How It Going
```
How it started: [humble beginning]
How it going: [current state — funny contrast]
```

### They Said It Was Impossible
```
[Person doing the impossible thing]
Caption: "They said [thing] was impossible"
```
