# Token Formats

## Solana Token
```
Token: $SYMBOL (Name)
Chain: Solana
CA: [base58 address]
```

## Ethereum / EVM Token
```
Token: $SYMBOL (Name)
Chain: Ethereum / BSC / Base
CA: 0x[hex address]
```

## Multi-Chain Token
```
Token: $SYMBOL (Name)
Chains: Solana, Ethereum, Base
Primary CA: [address] (Solana)
```

## New Launch
```
🆕 JUST LAUNCHED: $SYMBOL

CA: [address]
Chain: [chain]
DEX: [dex name]

MC: $[mc] (early!)
Liquidity: $[liquidity]

[Why it matters]

🔗 [dexscreener link]
🔗 [website]
🔗 [telegram]
```

## Established Token
```
📊 $SYMBOL Update

Price: $[price] ([%] 24h)
MC: $[mc]
Volume: $[volume]

Recent:
• [news item 1]
• [news item 2]

Still building. Still early.

$SYMBOL
```
