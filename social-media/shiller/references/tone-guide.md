# Tone Guide

## Degen Tone
- Heavy emoji: 🚀💎🙌🔥🦍
- All caps for emphasis
- Short, punchy sentences
- "WAGMI" / "GM" / "DEGEN" / "APED"
- "TO THE MOON" / "DIAMOND HANDS"
- "THIS IS THE ONE"

Example:
```
🚀🚀 $AGENT IS ABOUT TO EXPLODE 🚀🚀

CA: xxxxxxxxxx
MC: $500K → $5M easy

WAGMI ANONS 💎🙌

🔗 dexscreener.com/solana/xxx
```

## Professional Tone
- Clean, minimal formatting
- Data-driven
- No excessive emojis
- Focus on fundamentals
- Measured language

Example:
```
$AGENT — Building the future of AI agents on Solana.

📊 MC: $2M | Holders: 5,000
🔒 Liquidity: Locked 2 years
✅ Contract: Renounced
🌐 agent.xyz

Worth researching.
```

## Mysterious Tone
- Vague hints
- Countdown language
- Inside questions
- "Something is brewing"

Example:
```
Something big is coming to Solana.

You've been watching the wrong tokens.

The real alpha? It's not what you think.

Stay tuned. 🕐

#Solana #AI
```

## Community-Driven Tone
- "We" language
- Celebrate together
- Inclusive
- Milestone-focused

Example:
```
We just hit 10,000 holders! 🎉

This community is special:
✅ No bots
✅ No VCs
✅ Just builders

Welcome to the family, new anons.

We're just getting started. 💪

$AGENT
```
