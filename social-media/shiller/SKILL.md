# Shiller

Generate crypto token shill posts for X.

## When to Use

- Promoting a token or project
- Creating FOMO content
- Announcing new listings or partnerships
- Building community hype
- Responding to market movements

## Required Inputs

| Input | Required | Description |
|-------|----------|-------------|
| Token name/symbol | Yes | e.g., "Agent ($AGENT)" |
| Contract address | No | For verification links |
| Chain | No | Solana, ETH, BSC, Base, etc. |
| Tone | No | degen, professional, mysterious, community |
| CTA | No | Website, dexscreener, telegram, etc. |

## Tone Guide

See [references/tone-guide.md](references/tone-guide.md) for detailed tone examples.

### Degen
- Heavy emoji use (🚀💎🙌🔥)
- All caps phrases
- "WAGMI" / "GM" / "DEGEN" language
- Short, punchy sentences

### Professional
- Clean formatting
- Data-driven (MC, holders, volume)
- No excessive emojis
- Focus on fundamentals

### Mysterious
- Vague hints
- "Something big is coming"
- Countdown language
- Community inside jokes

### Community-Driven
- "We" language
- Celebrate milestones
- Highlight community wins
- Inclusive tone

## Procedure

1. Gather token info (name, symbol, CA, chain, MC, links)
2. Select tone based on audience
3. Choose template or create custom
4. Add relevant hashtags (3-5 max)
5. Include CTA link
6. Review before posting
7. Post via xurl

## Hashtag Strategy

- Use 3-5 hashtags max
- Mix broad (#crypto, #defi) with specific ($TOKEN, #chain)
- Trending hashtags when relevant
- Create branded hashtags for campaigns

## Safety Rules

- Never promise returns
- Always include "DYOR"
- Don't fake metrics
- Don't impersonate projects
- Follow X terms of service
- Disclose if paid promotion

## Verification

After posting:
- Confirm post appears on profile
- Check engagement after 1 hour
- Track which tones/templates perform best
