# Reply Templates

## Value-Add
```
This. And also [additional point].

[Specific example or data].
```

```
Great point. To add: [new information].

[Why it matters].
```

```
I've seen this too. What's interesting is [observation].

[Data or source if available].
```

## Question
```
Interesting. How do you think this affects [related topic]?

Curious to hear your take.
```

```
Love this. Question: [thoughtful follow-up]?

Would love to dig deeper.
```

```
This raises a good point. What about [edge case or counter]?

 Genuinely asking.
```

## Humor
```
This is the energy I needed today 😂

[Relatable addition].
```

```
Me: *reads this*
Also me: *screenshots for later*

Great stuff.
```

```
If this were a tweet, it would go viral.

Just saying. 👀
```

## Support
```
Couldn't agree more.

[Personal experience that aligns].
```

```
This is why I follow you.

Always bringing the heat 🔥
```

```
100% this.

Adding to the chorus: [amplification].
```

## Contrarian (Respectful)
```
Interesting take. I see it differently though:

[Alternative view with reasoning].

Respectfully disagree, but curious to hear more.
```

```
Counterpoint: [alternative perspective].

Not saying you're wrong, just adding nuance.
```

```
I used to think this too. Then [experience that changed my mind].

Now I believe [different view].
```
