# Thread Formats

## Educational Thread
```
Tweet 1 (Hook): "Most people don't understand [topic]. Here's the truth:"
Tweet 2: "First, let's talk about [concept]..."
Tweet 3: "The key insight is..."
Tweet 4: "Here's an example..."
Tweet 5: "This matters because..."
Tweet 6: "The bottom line:"
Tweet 7 (CTA): "If this helped, follow me @handle for more on [topic]"
```

## Narrative Thread
```
Tweet 1 (Hook): "6 months ago, I [situation]. Here's what happened:"
Tweet 2: "It started when..."
Tweet 3: "Then things got interesting..."
Tweet 4: "The turning point was..."
Tweet 5: "I learned that..."
Tweet 6: "Fast forward to today..."
Tweet 7 (CTA): "The lesson: [takeaway]. Repost if this resonated 🔄"
```

## Listicle Thread
```
Tweet 1 (Hook): "10 [topic] tips that changed my [outcome]:"
Tweet 2: "1/ [Tip 1]"
Tweet 3: "2/ [Tip 2]"
Tweet 4: "3/ [Tip 3]"
...
Tweet N (CTA): "Which one hit hardest? Reply below 👇"
```

## Hot Take Thread
```
Tweet 1 (Hook): "Unpopular opinion: [controversial take]"
Tweet 2: "Here's why..."
Tweet 3: "The data shows..."
Tweet 4: "But here's what people miss..."
Tweet 5: "I could be wrong, but..."
Tweet 6 (CTA): "Agree or disagree? Let me know 👇"
```

## Build in Public Thread
```
Tweet 1 (Hook): "I'm building [project]. Here's my progress:"
Tweet 2: "Week 1: [what I did]"
Tweet 3: "Week 2: [what I did]"
Tweet 4: "Challenges I faced:"
Tweet 5: "What I learned:"
Tweet 6: "Next steps:"
Tweet 7 (CTA): "Follow along for weekly updates 🚀"
```

## Comparison Thread
```
Tweet 1 (Hook): "I compared [A] vs [B] for 30 days. The results:"
Tweet 2: "Setup: [how I tested]"
Tweet 3: "Metric 1: [A] won because..."
Tweet 4: "Metric 2: [B] won because..."
Tweet 5: "Metric 3: Tie because..."
Tweet 6: "The winner: [verdict]"
Tweet 7 (CTA): "Which one are you using? 👇"
```
