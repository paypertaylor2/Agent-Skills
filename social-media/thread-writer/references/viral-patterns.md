# Viral Patterns

## What Makes Threads Go Viral

### 1. The Hook (Most Important)
- Stop the scroll in 3 seconds
- Create curiosity gap
- Challenge a belief
- Promise value
- Use numbers ("10 tips", "5 mistakes")

### 2. Pattern Interrupt
- Break expectations
- "Here's what nobody tells you..."
- "I was wrong about..."
- "The opposite of what you think..."

### 3. Value Density
- Every tweet must earn the next one
- No filler
- Each tweet = one idea
- Build momentum

### 4. Emotional Arc
- Start with tension
- Build curiosity
- Deliver payoff
- End with CTA

### 5. Formatting
- Short lines
- Line breaks between ideas
- Bold key phrases
- Number tweets (1/ 2/ 3/)
- White space

## Hook Formulas

### Curiosity
- "I spent 100 hours researching [topic]. Here's what I found:"
- "The [topic] secret nobody talks about:"
- "Why [common belief] is wrong:"

### Authority
- "After [X years] in [field], here's what I know:"
- "I've [achievement]. Here's how:"
- "As a [role], I can tell you:"

### Contrarian
- "Unpopular opinion:"
- "Hot take:"
- "Everyone says [X]. I say [Y]."

### Story
- "6 months ago, [situation]. Now [result]."
- "I almost [failure]. Then [turning point]."
- "The day everything changed:"

### List
- "10 [things] that [outcome]:"
- "5 [mistakes] to avoid:"
- "7 [tips] for [goal]:"

## Engagement Triggers

- Ask questions
- Request reposts
- Create polls
- Tag relevant accounts
- Use "Reply below 👇"
- Use "Save this for later 🔖"

## Timing

- Best days: Tuesday, Wednesday, Thursday
- Best times: 9-11 AM, 1-3 PM, 5-7 PM EST
- Post threads when your audience is active
- Avoid weekends for educational content
