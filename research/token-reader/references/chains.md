# Supported Chains

## Solana
- Address format: Base58 (e.g., `So11111111111111111111111111111111111111112`)
- DEXes: Raydium, Orca, Jupiter, Meteora
- Block time: ~400ms
- Explorer: solscan.io, solscan.io

## Ethereum
- Address format: Hex (e.g., `0x...`)
- DEXes: Uniswap, SushiSwap, Curve, Balancer
- Block time: ~12s
- Explorer: etherscan.io

## BSC (BNB Chain)
- Address format: Hex (same as ETH)
- DEXes: PancakeSwap, BiSwap
- Block time: ~3s
- Explorer: bscscan.com

## Base
- Address format: Hex (same as ETH)
- DEXes: Aerodrome, Uniswap, BaseSwap
- Block time: ~2s
- Explorer: basescan.org

## Polygon
- Address format: Hex (same as ETH)
- DEXes: QuickSwap, Uniswap
- Block time: ~2s
- Explorer: polygonscan.com

## Arbitrum
- Address format: Hex (same as ETH)
- DEXes: Uniswap, GMX, Camelot
- Block time: ~0.25s
- Explorer: arbiscan.io

## Avalanche
- Address format: Hex (same as ETH)
- DEXes: Trader Joe, Pangolin
- Block time: ~2s
- Explorer: snowtrace.io

## Optimism
- Address format: Hex (same as ETH)
- DEXes: Velodrome, Uniswap
- Block time: ~2s
- Explorer: optimistic.etherscan.io
