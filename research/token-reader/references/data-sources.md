# Data Sources

## DexScreener
- **URL:** dexscreener.com
- **API:** https://api.dexscreener.com/latest/dex/tokens/{tokenAddress}
- **Data:** Price, volume, liquidity, pairs, FDV, pair age
- **Chains:** 70+ chains
- **Rate limit:** 300 requests per minute
- **Auth:** None required

## Birdeye
- **URL:** birdeye.so
- **API:** https://public-api.birdeye.so/public/...
- **Data:** Price, OHLCV, market cap, holders, trading history
- **Chains:** 20+ chains
- **Rate limit:** Varies by plan
- **Auth:** API key required (free tier available)

## DexTools
- **URL:** dextools.io
- **API:** https://www.dextools.io/shared/api/...
- **Data:** Charts, holder analysis, score
- **Chains:** ETH, BSC, Polygon, and more
- **Rate limit:** Varies
- **Auth:** API key for advanced features

## Solscan
- **URL:** solscan.io
- **API:** https://public-api.solscan.io/...
- **Data:** On-chain transactions, token info, account info
- **Chain:** Solana only
- **Rate limit:** 10 requests per second
- **Auth:** None for basic, API key for advanced

## Etherscan
- **URL:** etherscan.io
- **API:** https://api.etherscan.io/api
- **Data:** Transactions, contract info, token info
- **Chains:** Ethereum and EVM chains (each has own explorer)
- **Rate limit:** 5 calls/second
- **Auth:** API key required (free)

## Jupiter (Solana)
- **URL:** jup.ag
- **API:** https://quote-api.jup.ag/v6/quote
- **Data:** Swap quotes, routes
- **Chain:** Solana only
- **Rate limit:** Reasonable use
- **Auth:** None required
